[![License: MIT](https://img.shields.io/badge/License-MIT-0F172A.svg)](./LICENSE)  Personal portfolio built with **React 19 + TypeScript + Vite + Tailwind CSS v4**.  **Demo:** [julio-cesar-morales.netlify.app](https://julio-cesar-morales.netlify.app)  ## Features  - Modern, responsive design - Solid baseline accessibility - Technical SEO and complete metadata - GitHub stats with caching - Contact form with Formspree - Structure ready to keep growing  ## Tech stack  | Category | Technology | |-----------|-----------| | Framework | React 19 | | Language | TypeScript 5 | | Build tool | Vite | | Styling | Tailwind CSS v4 | | Animations | Framer Motion | | Icons | React Icons | | Deploy | Netlify / Vercel |  ## Installation  ```bash git clone https://github.com/Victor00128/julio-cesar-portfolio.git cd julio-cesar-portfolio npm install npm run dev ```  The project will be available at `http://localhost:5173`.  ## Environment variables  The repository includes `.env.example`.  - `VITE_FORMSPREE_URL`: optional, if you want to override the form's current endpoint - `VITE_GA_ID`: optional, for analytics - `VITE_GITHUB_URL`, `VITE_LINKEDIN_URL`, `VITE_EMAIL`: optional overrides  ```bash cp .env.example .env ```  ## Production deploy  ### Netlify  1. Push the code to GitHub. 2. Import the repository in Netlify. 3. Use `npm run build` as the build command. 4. Use `dist` as the publish directory.  ### Vercel  1. Run `vercel`. 2. Follow the CLI instructions.  ## Project structure  ```text src/ ├── components/ ├── hooks/ ├── App.tsx ├── index.css └── main.tsx public/ └── CV-Julio-Cesar.pdf ```  ## Available commands  ```bash npm run dev npm run build npm run preview ```  ## Notes  - If you fork the project, change the contact details and the Formspree endpoint. - The CV preparation instructions were moved to `docs/CV-INSTRUCTIONS.md`.  ## License  MIT.
